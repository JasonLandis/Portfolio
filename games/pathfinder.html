Overview

Pathfinder was one of my first projects. It is currently a prototype but I plan to work on it more in the future.

How to play

You are given a map with a start tile and an end tile. On the start tile, there is a character that searches for the most optimal path to the end tile. Once you hit the play button, the character begins to move towards the end tile. The goal of the game is to prevent the character from reaching the end tile within a certain amount of time. You can stall the character by placing blockades causing it to reroute its path.
