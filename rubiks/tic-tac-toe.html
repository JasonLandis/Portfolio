Tic-Tac-Toe Cube

Each square in this puzzle is either an X or an O. There are many ways you can solve this, but some I have not proven yet.

Solved States

1. X or O is winning on each side
2. All X's on the same 3 sides and All O's on the same 3 sides

Potential Solved States

1. Only X is winning on each side
2. Only O is winning on each side
3. Neither X nor O is winning on every side
